# v0.1.0 人工验收计划

## 目标

在 `v0.1.0` 便携版（`JvedioNext_0.1.0_x64-portable.zip`）上完成一轮**端到端人工验收**，覆盖从解压启动到日常使用的全部核心流程，确认真实用户体验无阻断性问题后，再启动下一版功能规划。

## 背景

- Phase 10 自动化 E2E 已跑通 7 组 flow（Playwright MCP + 浏览器模式）
- 62 个后端测试全部通过
- 但自动化验收运行在浏览器模式，**尚未在正式安装包环境下做过人工全链路验证**
- 本次验收的目标环境是：**安装 NSIS 安装包 → 双击 `JvedioNext.exe` → 真实 Tauri 壳层**

## 验收环境

| 项目 | 要求 |
|------|------|
| 安装包 | `build/release/JvedioNext_0.1.0_x64-setup.exe` |
| 运行环境 | Windows 10/11 x64 |
| 前置条件 | 无需手动启动 Worker（Tauri Shell 自动拉起） |
| 测试数据 | 准备 4+ 个真实影片文件用于扫描导入 |
| MetaTube | 需要可用的 MetaTube 实例（用于搜刮验证） |

## 验收通过标准

1. Phase 1–3（解压启动/导航/库管理）零 P0/P1 问题
2. Phase 4–5（扫描/搜刮）核心链路跑通，有成功和失败样本
3. Phase 6–8（浏览/详情/收藏/演员）主要流程无阻断
4. Phase 9（设置）保存/恢复正常
5. Phase 10（重抓/删除/外链）关键操作可执行
6. 所有发现的问题已记录并分级
