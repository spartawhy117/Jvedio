# Series Page Spec

## 页面目的

- `Series Page` 是智能分类中的全局系列聚合页。
- 它负责展示系列列表，并在选中某个系列后展示该系列下的影片结果集。

## 页面范围

- 本页负责：
  - 展示全局系列列表
  - 支持系列关键字筛选
  - 支持系列结果排序
  - 展示选中系列下的影片结果集
  - 从系列结果进入影片详情页
- 本页不负责：
  - 单库范围的系列管理
  - 系列元数据编辑
  - 媒体库管理动作

## 数据来源

- 系列聚合列表：
  - `GET /api/videos/series`
- 系列下影片结果集：
  - `GET /api/videos/series/{seriesName}/videos`
- 影片详情：
  - `GET /api/videos/{videoId}`

## 布局

- 本页复用 `main-shell.md` 定义的共享壳层。
- 当前线框只表达右侧内容区。
- 顶部保留：
  - 搜索框
  - 排序按钮组
- 内容区采用左右分栏：
  - 左侧为系列列表
  - 右侧为当前选中系列的影片结果区

## 元素清单

| 元素 | 类型 | 常显 | 行为 | 数据来源 | 说明 |
| --- | --- | --- | --- | --- | --- |
| 页面标题 | 静态标题 | 是 | 无 | 固定文案 | 文案为 `系列 / Series` |
| 搜索框 | 输入框 | 是 | 更新关键字筛选 | 当前查询状态 | 可筛系列名或结果关键字 |
| 排序按钮组 | 轻量切换按钮 | 是 | 切换排序字段 | 当前查询状态 | 第一轮保留数量/最近扫描 |
| 系列列表 | 列表区 | 是 | 切换当前系列 | `GET /api/videos/series` | 左侧聚合列表 |
| 系列列表项 | 列表项 | 是 | 选中后刷新右侧结果 | 当前聚合项 | 显示名称和影片数 |
| 系列结果标题 | 标题+计数 | 是 | 无 | 当前选中系列 | 显示当前系列名和影片总数 |

| 分页控件 | 翻页控件 | 是 | 切换当前页 | 当前查询状态 | 位于右下角 |

## 交互规则

- 从左侧 `智能分类 > 系列` 进入本页。
- 本页默认是全局聚合，不限定某个媒体库。
- 选中左侧系列项后，右侧展示该系列的影片结果。
- 点击影片卡片后进入影片详情页。
- 从影片详情返回时应恢复当前系列名、筛选、排序和分页状态。

## 状态定义

### Loading

- 首次进入本页时显示系列列表或结果区 loading。

### Empty

- 没有系列结果时显示空态。
- 当前系列下没有影片时，右侧结果区显示空态。

### Error

- 系列列表或影片结果拉取失败时显示错误提示与刷新入口。

## 性能与体验约束

- 左侧系列列表和右侧影片结果区职责分离。
- 系列页保持与 `categories-page` 相同的聚合页骨架。
- 页面图只保留结构与控件，不写说明性长文本。

## 回归点

- 能从左侧导航进入 Series。
- 系列列表能正确显示。
- 选中系列后右侧影片结果能正确刷新。
- 筛选、排序、分页有效。
- 进入影片详情并返回后状态恢复正常。
