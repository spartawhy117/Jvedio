# Library Page Spec

## Purpose

- Show the contents of the selected media library.

## Layout

- Page title uses the selected library name.
- Top action row:
  - search
  - view mode toggle
  - filter entry
  - edit or batch mode entry when applicable
- Main content:
  - current video list content area
  - optional secondary filter panel
  - details area remains governed by existing interaction patterns

## Behavior

- Selecting a library from the left navigation opens this page.
- Existing browsing, filtering, search, and detail-opening behavior should be preserved in the later implementation phase.

## Notes

- This page is distinct from Home.
- Home manages libraries.
- Library Page browses a selected library's contents.
