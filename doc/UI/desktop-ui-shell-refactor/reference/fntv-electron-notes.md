# `fntv-electron` 参考说明

## 参考项目

- 仓库：
  - [QiaoKes/fntv-electron](https://github.com/QiaoKes/fntv-electron)

## 当前已确认的参考点

- 桌面壳思路：
  - Electron 作为桌面前端容器
- 页面组织：
  - Web 前端承载主要页面内容
- 桌面交互感：
  - 强调本地客户端体验而不是浏览器站点感
- 页面密度：
  - 偏桌面工具型信息密度，而非大留白展示型布局

## 明确不纳入 Jvedio 第一阶段的能力

- 多账户管理
- 远程访问
- MPV 深度增强
- 弹幕
- 智能跳过
- 硬解专项能力
- 跨平台分发目标

## 复用边界

- 允许：
  - 参考桌面壳节奏
  - 参考导航组织
  - 参考页面内容密度
