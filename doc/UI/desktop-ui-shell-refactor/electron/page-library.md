# Library 页规格

## 页面定位

- Library 页展示单个媒体库下的影片内容。
- 也是扫描与抓取闭环的主要入口页。

## 主要能力

- 浏览影片列表或卡片
- 排序
- 分页
- 筛选
- 进入影片详情
- 发起扫描任务
- 发起抓取任务

## 页面结构

- 顶部：
  - 库名
  - 排序控件
  - 扫描按钮
  - 抓取按钮
- 主体：
  - 影片列表或卡片网格
- 底部或侧边：
  - 分页信息
  - 任务摘要入口

## 第二阶段重点

- 配置默认扫描目录后能发起扫描。
- 命中影片后整理、抓取与输出结果可反馈到当前库视图。
- 未命中影片必须有明确报告。

## 数据来源

- 后续由 Worker 提供：
  - libraries API
  - scan API
  - scrape API
  - tasks API
