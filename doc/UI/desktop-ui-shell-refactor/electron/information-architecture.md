# Electron 信息架构

## 主壳

- 固定左导航
- 自适应右内容区
- 设置作为壳层入口
- 不做浏览器地址栏感

## 左侧导航

- Home
- Favorites
- Actors
- 智能分类
  - 类别
  - 系列
- Libraries
  - 每个库一项

## 页面职责

### Home

- 媒体库管理中心
- 新建库
- 删除库
- 重命名库
- 打开库
- 编辑默认扫描目录入口

### Favorites

- 收藏聚合页
- 第一阶段沿用现有聚合语义，不扩展新业务规则

### Actors

- 演员聚合一级页
- 搜索
- 排序
- 查看演员关联影片

### Library

- 单个媒体库内容浏览
- 扫描与抓取入口
- 分页、排序、筛选
- 进入影片详情

### 类别 / 系列

- 聚合浏览页
- 第一阶段保持只读浏览，不新增复杂编辑能力

### Video Detail

- 影片详情
- 演员、图片、路径、元数据展示
- 播放入口

### Settings

- 壳层入口打开
- 承接现有设置能力
- 语言与主题仅公开：
  - 中文 / English
  - Light / Dark
