# Video Detail 页规格

## 页面定位

- 展示单个影片的详情信息。
- 承接播放入口。

## 主要能力

- 展示基础元数据
- 展示演员信息
- 展示图片与路径
- 打开外部链接
- 调用播放

## 页面结构

- 顶部：
  - 标题
  - VID
  - 主要操作按钮
- 主体：
  - 海报与背景图
  - 基础信息
  - 演员区域
  - 文件路径与来源信息

## 第三阶段重点

- 播放仍走当前外部播放器链。
- 不做内嵌播放器。
- 播放成功后要维持现有写回语义。

## 数据来源

- 当前视频实体与详情窗口相关能力
- 后续由 Worker videos API 统一提供
