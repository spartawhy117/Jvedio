# Settings 页规格

## 页面定位

- Settings 由主壳入口打开。
- 承接当前 `Window_Settings` 的设置能力。

## 第一阶段分组

- 基本
- 图片
- 扫描与导入
- 网络
- 库
- MetaTube
- 版本检查

## 第一阶段规则

- 第一阶段 UI 仅公开：
  - 中文 / English
  - Light / Dark
- 保留保存、应用、恢复默认语义。
- 不新增首启主题/语言向导。

## 重点设置项

- 视频播放器路径
- MetaTube 相关配置
- 扫描导入配置
- 图片与缓存相关配置
- 版本检查入口

## 第四阶段重点

- 展示当前值
- 保存后可回读
- 设置变更能被后续播放、扫描与抓取链消费
