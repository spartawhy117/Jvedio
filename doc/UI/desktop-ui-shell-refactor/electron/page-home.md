# Home 页规格

## 页面定位

- Home 是媒体库管理页，不是封面墙首页。
- 本页承接旧 `WindowStartUp` 的媒体库 CRUD 与打开动作。

## 主要能力

- 新建库
- 删除库
- 重命名库
- 打开库
- 查看库基础状态
- 进入库扫描路径编辑入口

## 页面结构

- 顶部：
  - 页面标题
  - 新建库按钮
- 主体：
  - 媒体库列表
  - 每行展示：
    - 库名
    - 路径摘要
    - 状态摘要
    - 操作按钮

## 第一阶段重点

- 先完成新建和删除闭环。
- 不在本页扩展复杂封面管理。
- 不在本页承接扫描任务详情展示，任务详情另由任务中心或 Library 页承接。

## 数据来源

- 当前库配置与数据库记录
- 后续由 Worker libraries API 统一提供
