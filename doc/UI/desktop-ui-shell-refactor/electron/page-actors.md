# Actors 页规格

## 页面定位

- Actors 是左侧一级导航重点页。
- 负责全库演员聚合浏览，而不是某个库的局部演员页。

## 主要能力

- 搜索演员
- 按名字排序
- 按影片数量排序
- 展示演员头像、名字、影片数量
- 进入演员关联影片列表或详情

## 页面结构

- 顶部：
  - 页面标题
  - 搜索框
  - 排序控件
- 主体：
  - 演员卡片网格

## 第一阶段重点

- 保留现有演员聚合与搜索语义。
- 不扩展演员编辑能力。
- 不扩展复杂收藏或关系编辑能力。

## 数据来源

- 当前 `ActorList` 与演员相关聚合逻辑
- 后续由 Worker actors API 统一提供
